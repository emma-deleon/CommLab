                                                        |                     |
                                                        |  Step 2: Master Your|
                                                        |           State of  |
                                                        |            Being    |
                                                        |                     |
                                                        | In your first few   |
                                                        | days, you may find  |
                                                        | yourself            |                                                                                    |                     |
                                                        | accidentally        |
                                                        | drifting through    |
                                                        | walls or sinking    |
                                                        | into the floor.     |
                                                        |                     |
                                                        | This is both        |
                                                        | undignified and     |
                                                        | inefficient.        |
                                                        |                     |
                                                        | Practice!:          |
                                                        |                     |
                                                        |   Focus on a solid  |
                                                        |   object, like a    |
                                                        |   table. Visualize  |
                                                        |   your "weight" and |
                                                        |   "solidity."       |
                                                        |                     |
                                                        |   Attempt to simply |
                                                        |   rest on the       |
                                                        |   surface.          |
                                                        |                     |
                                                        |   Don't try to      |
                                                        |   move it yet!      |
                                                        |                     |
                                                        |   Just exist next   |
                                                        |   to it.            |
                                                        |                     |
                                                        | Practice this for   |
                                                        | a few hours         |
                                                        | (you have the time) |
                                                        |                     |
